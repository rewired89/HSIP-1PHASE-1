**HSIP Governance**

HSIP (Hyper-Secure Internet Protocol) is an open-source project developed and stewarded by Nyx Systems LLC.
This governance document explains how decisions are made, how contributions work, and how the project remains stable and sustainable.

**1. Stewardship Model**
Project Steward: Nyx Systems LLC
Lead Maintainer: Rewired89
Email: nyxsystemsllc@gmail.com

Nyx Systems LLC provides long-term leadership, vision, and commercial support for HSIP.
The project follows a Commercial Stewardship Model, where:
The protocol is open-source
The project is community-friendly
The direction and long-term decisions remain under Nyx Systems LLC
This ensures consistency, security, and a clear path to sustainability.

**2. Decision-Making**
Handled by the project lead or delegated maintainers
Discussed publicly on GitHub (Issues or Discussions)
Community feedback is welcome
Final decision is made by the Lead Maintainer

**HSIP’s direction must remain cohesive and secure, therefore, final authority stays with Nyx Systems LLC.**

**Security issues**
Handled privately and urgently.
Fixes are developed in private, then disclosed responsibly.

**3. Contribution Process**
New features
Start a GitHub Discussion first.
After alignment, submit a PR.

Require a design proposal (RFC-style discussion).
Maintainer approval is required before implementation.

**4. Roles**
Project Lead
Responsible for:
Protocol design
Release management
Security decisions
Roadmap
Licensing and legal

**Future Maintainers**
Nyx Systems LLC may invite additional maintainers based on:
Consistent contributions
Technical quality
Professional conduct
Maintainers assist with reviews and small releases, but strategic decisions remain with Nyx Systems LLC.

**Contributors**
Anyone can submit PRs, report bugs, and participate in discussions.

**Users**
Provide feedback, request features, and help spread awareness.

**5. Transparency**
Public:
Issues
Discussions
Pull requests
Release notes
Roadmap (high-level only)

**6. Intellectual Property**
Code is open-source under the HSIP Community License (non-commercial).
Commercial use requires an Enterprise License from Nyx Systems LLC.
“HSIP” and related trademarks are owned by Nyx Systems LLC.
This ensures the protocol remains open while enabling sustainable commercial support.

**7. Changes to Governance**
The governance model may evolve as the community grows, but must remain compatible with long-term sustainability.
Nyx Systems LLC retains final authority over governance updates.
